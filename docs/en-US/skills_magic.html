SPELL LIST


Your characters will learn a number of spells throughout the game. This is but a sample of them.

Spell List
Diamond Dust
Attack magic which freezes the ambient water vapor in the air into massive crystals above the caster and sends them smashing down upon foes. Can freeze foes.
→ Type Damage
→ Element Water
→ Cast Time Long
Heal Water
Soothes injuries with holy water.
→ Type Restorative
→ Element Water
→ Cast Time Long
Inferno
Sets a large patch of ground in front of the caster ablaze for a time. Foes within the blaze take constant damage until they get out.
→ Type Damage
→ Element Fire
→ Cast Time Medium
Shield
Creates a magical shield around the target. Shielded targets take half damage and do not suffer spell interruption. As the target takes damage, the shield fades.
→ Type Support
→ Element None
→ Cast Time Short
Disintegrate
Emits an aura of destruction in all directions, vaporizing any foes touched by it. Does not work on targets stronger than the caster.
→ Type Damage
→ Element None
→ Cast Time Short
Mirror Image
Creates two mirror images of an ally. The copies replicate all physical attacks made by the target.
→ Type Support
→ Element None
→ Cast Time Short
Sleep Mist
Creates a cloud of fog which will put enemies within to sleep. Sleeping enemies will continue to sleep until they take substantial amounts of damage.
→ Type Support
→ Element None
→ Cast Time Short
Chaser
Launches balls of fire at foes. The balls will chase their target and are very accurate.
→ Type Damage
→ Element Fire
→ Cast Time Long
Teleportation
Returns the caster and her allies to a safe location. However, some areas do not allow teleportation. Can be used in combat, but the cast time is exceptionally long (making the risk of interruption high).
→ Type Support
→ Element None
→ Cast Time Extreme
Interceptor
Fires countless flaming arrows upward from the ground to strike enemies, including airborne ones.
→ Type Damage
→ Element Fire
→ Cast Time Short
Water Blow
Sends forth a massive wave of water in front of the caster, striking all enemies multiple times.
→ Type Damage
→ Element Water
→ Cast Time Long
Explosion
Sends forth a ball of flame which explodes in a tremendous fashion, dealing staggering damage to any enemies in a huge area.
→ Type Damage
→ Element Fire
→ Cast Time Long
