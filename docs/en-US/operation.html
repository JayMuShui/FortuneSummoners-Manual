CONTROLS
Core Keyboard Controls
1 Arrow Keys
→ Left and Right move your character in the given direction; pressing twice makes you dash, pressing the opposite direction while dashing allows you to stop quickly.
→ Up guards against enemy attacks.
→ Up also allows you to enter doors and gates when in front of them.
→ Down makes your character duck.
→ Up and down also allow you to scroll through menus.
2 C Key
→ Jump
→ Cancel menu
X Key
→ Attack
→ Hold & release to cast spells (magicians)
→ Talk/Search
→ Select Menu Option
Z Key
→ Draw/sheath sword
→ Select active spell (magicians)
→ Hold and use arrow keys to quickly select magic
3 Enter Key
→ Change controlled character in a party
4 F1 Key
→ Opens the main menu
F2 Key
→ Use a healing item on the most injured character




Other Keyboard Controls
1 TAB Key
→ Skip messages/dialogue
2 F3 Key
→ Cycle through party member follow modes
F4 Key
→ Cycle through party member magic use mdoes
F5 Key
→ Change active magic book
F6 Key
→ Change scrolling mode
3 F8 Key
→ Reset
→ Press on title screen to quit
4 0ｽｿｽｽ`9 Keys
→ Select the associated magic spell, number bar and pad both function
5 Pause Key
→ Pause the game
6 Home Key
End Key
PageUp Key
PageDown Key
→ Home: Scrolls screen left
→ End: Scrolls screen right
→ PageUp: Scrolls screen up
→ PageDown: Scrolls screen down
- F11 Key
→ Take screenshot (images will be placed in user folder)
- ALT + F4
→ Emergency quit to Windows
- ALT + TAB
→ Switch applications (not recommended in fullscreen mode)

Keyboard Customization
Keybinds may be changed at any time in the "Keyboard Settings" option in the Options menu.

Gamepad Customization
As gamepad layouts can vary between manufacturers & models, we recommend double-checking and optimizing pad bindings in the options menu under "Gamepad Settings".
Up to sixteen face buttons are supported, as well as full function mapping for both HAT switch-style d-pads and right-hand sticks.
