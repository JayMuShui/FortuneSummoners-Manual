STARTING THE GAME


To launch the game, run sotes.exe from the Fortune Summoners installation folder. (The Steam version may be run from the Steam browser.)



You will see an options menu with available graphics options when starting the game. Select your preferred options and select "launch game".
The default (and recommended) options are displayed below.

Configuring Graphics Quality
If you find the game runs slowly with default settings, set the graphics quality to "low" and VRAM usage to "all". (Using all available VRAM can cause slowdown in other cases, so if the game runs smoothly on default settings, don't change this option.)

If screen changes happen too quickly at default settings causing the screen to cut, set VRAM Usage to "Partial" or "Full."

Most users will find the best visual quality and highest framerate comes from "Visual Quality = High" and "VRAM Usage = Partial."

Once the game is launched, the title menu will have the following options.


Start Start the game from the beginning.
Continue Load saved data and continue the game from a previous save.
Bonus Menu "VS Mode" and "Auto-play Mode" bonuses and options unlocked during play are in this menu.
Options Allows you to configure game settings.
Exit Exits to Windows.
