开始游戏


要启动游戏，请运行《命运召唤 尔茄的精灵石》安装目录下的 sotes.exe 文件。（Steam 版本可通过 Steam 客户端启动。）



启动游戏时，你会看到一个包含各类图形设置选项的配置菜单。选择你偏好的设置后，点击「启动游戏」即可。
下方展示的是默认（且推荐使用）的设置选项。

图形质量配置说明
如果使用默认设置时游戏运行卡顿，可将图形质量设为「低」，并将显存占用设为「全部」。（需要注意的是，占用全部可用显存可能会在其他场景下导致运行卡顿，因此如果默认设置下游戏运行流畅，请勿修改该选项。）

如果默认设置下画面切换过快导致画面卡顿撕裂，可将显存占用设为「部分」或「全部」。

对于大多数玩家而言，将「画面质量 = 高」与「显存占用 = 部分」搭配使用，能获得最佳的画面效果和最高的帧率。

游戏成功启动后，标题界面将显示以下选项。


新游戏（Start） 从头开始进行游戏。
继续游戏（Continue） 读取存档数据，从之前的存档点继续游戏。
奖励菜单（Bonus Menu） 游戏过程中解锁的「对战模式」「自动游玩模式」等奖励内容及相关选项均在此菜单中。
设置（Options） 可在此配置各类游戏相关设置。
退出（Exit） 退出游戏，返回 Windows 桌面。
