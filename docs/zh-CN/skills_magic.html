魔法列表


游戏中，你的角色将习得多种魔法。以下仅为部分示例。

魔法列表
钻石尘埃（Diamond Dust）
将空气中的水蒸气冻结成巨大冰晶，在施法者上方凝聚后砸向敌人的攻击魔法。可使敌人冻结。
→ 类型（Type） 伤害型（Damage）
→ 元素（Element） 水（Water）
→ 施法时间（Cast Time） 长（Long）
治愈之水（Heal Water）
用圣水抚慰伤痛，恢复生命值。
→ 类型（Type） 治愈型（Restorative）
→ 元素（Element） 水（Water）
→ 施法时间（Cast Time） 长（Long）
炼狱之火（Inferno）
使施法者前方大范围地面燃起持续火焰。火焰范围内的敌人会持续受到伤害，直至离开火焰区域。
→ 类型（Type） 伤害型（Damage）
→ 元素（Element） 火（Fire）
→ 施法时间（Cast Time） 中（Medium）
护盾（Shield）
在目标周围形成魔法护盾。护盾状态下的目标受到的伤害减半，且不会被打断施法。随着目标受到伤害，护盾会逐渐消散。
→ 类型（Type） 辅助型（Support）
→ 元素（Element） 无（None）
→ 施法时间（Cast Time） 短（Short）
湮灭（Disintegrate）
向四周释放毁灭气场，将触及的敌人化为灰烬。对等级高于施法者的目标无效。
→ 类型（Type） 伤害型（Damage）
→ 元素（Element） 无（None）
→ 施法时间（Cast Time） 短（Short）
镜像分身（Mirror Image）
为队友创造两个镜像分身。分身会复制目标的所有物理攻击。
→ 类型（Type） 辅助型（Support）
→ 元素（Element） 无（None）
→ 施法时间（Cast Time） 短（Short）
睡眠迷雾（Sleep Mist）
制造一片迷雾，使范围内的敌人陷入睡眠状态。睡眠中的敌人在受到大量伤害前不会苏醒。
→ 类型（Type） 辅助型（Support）
→ 元素（Element） 无（None）
→ 施法时间（Cast Time） 短（Short）
追踪火球（Chaser）
向敌人发射火球，火球会自动追踪目标，命中率极高。
→ 类型（Type） 伤害型（Damage）
→ 元素（Element） 火（Fire）
→ 施法时间（Cast Time） 长（Long）
传送术（Teleportation）
将施法者及其队友传送至安全区域。但部分区域不允许使用传送术。战斗中可使用，但施法时间极长（被打断的风险极高）。
→ 类型（Type） 辅助型（Support）
→ 元素（Element） 无（None）
→ 施法时间（Cast Time） 极长（Extreme）
截击箭雨（Interceptor）
从地面向上发射无数火焰箭矢，可攻击包括空中敌人在内的所有目标。
→ 类型（Type） 伤害型（Damage）
→ 元素（Element） 火（Fire）
→ 施法时间（Cast Time） 短（Short）
水之冲击（Water Blow）
向施法者前方释放巨大水浪，对范围内所有敌人造成多次打击。
→ 类型（Type） 伤害型（Damage）
→ 元素（Element） 水（Water）
→ 施法时间（Cast Time） 长（Long）
爆炎术（Explosion）
发射一枚火焰球，落地后产生巨大爆炸，对大范围区域内的敌人造成毁灭性伤害。
→ 类型（Type） 伤害型（Damage）
→ 元素（Element） 火（Fire）
→ 施法时间（Cast Time） 长（Long）
