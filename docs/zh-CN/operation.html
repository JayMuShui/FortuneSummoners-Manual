操作说明
核心键盘控制
1 方向键
→ 左右键控制角色向对应方向移动；双击可冲刺，冲刺时按反方向键可快速急停。
→ 上键可防御敌人攻击。
→ 站在门前时，上键可进入门或大门。
→ 下键可让角色下蹲。
→ 上下键还可用于滚动菜单。
2 C键
→ 跳跃
→ 取消菜单
X键
→ 攻击
→ 魔法师长按后松开可施法
→ 对话/调查
→ 选择菜单选项
Z键
→ 拔剑/收剑
→ 魔法师选择当前激活的魔法
→ 长按并配合方向键可快速选择魔法
3 回车键
→ 切换小队中当前操控的角色
4 F1键
→ 打开主菜单
F2键
→ 使用治疗道具治疗伤势最重的角色




其他键盘控制
1 TAB键
→ 跳过对话/文本
2 F3键
→ 循环切换小队成员的跟随模式
F4键
→ 循环切换小队成员的魔法使用模式
F5键
→ 切换当前激活的魔法书
F6键
→ 切换视角滚动模式
3 F8键
→ 重置游戏
→ 标题画面按下可退出游戏
4 数字键0-9
→ 选择对应的魔法，主键盘数字区和小键盘均可使用
5 Pause键
→ 暂停游戏
6 Home键
End键
PageUp键
PageDown键
→ Home键：向左滚动屏幕
→ End键：向右滚动屏幕
→ PageUp键：向上滚动屏幕
→ PageDown键：向下滚动屏幕
- F11键
→ 截图（图片将保存至用户文件夹）
- ALT + F4
→ 紧急退出至Windows系统
- ALT + TAB
→ 切换应用程序（全屏模式下不建议使用）

键盘按键自定义
可随时在选项菜单的「键盘设置」中修改按键绑定。


手柄控制
方向键/摇杆
→ 左右方向控制角色向对应方向移动；双击可冲刺，冲刺时按反方向可快速急停。
→ 上方向可防御敌人攻击。
→ 站在门前时，上方向可进入门或大门。
→ 下方向可让角色下蹲。
→ 上下方向还可用于滚动菜单。
按钮0
→ 使用治疗道具治疗伤势最重的角色
按钮1
→ 跳跃
→ 取消菜单
按钮2
→ 攻击
→ 魔法师长按后松开可施法
→ 对话/调查
→ 选择菜单选项
按钮3
→ 拔剑/收剑
→ 魔法师选择当前激活的魔法
→ 长按并配合方向键可快速选择魔法
按钮4
→ 切换当前激活的魔法书
按钮5
→ 直接选择1号魔法
按钮6
→ 长按该按钮并推动方向键/摇杆，可向对应方向滚动屏幕
按钮7
→ 直接选择2号魔法
按钮8
→ 切换小队中当前操控的角色
→ 跳过对话/文本
按钮9
→ 打开主菜单

手柄按键自定义
不同厂商和型号的手柄布局可能存在差异，建议在选项菜单的「手柄设置」中仔细检查并优化按键绑定。
手柄支持最多16个正面按钮，同时完全支持十字键式方向键和右侧摇杆的功能映射。
