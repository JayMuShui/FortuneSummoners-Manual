剑术技能列表


在冒险旅程中，尔茄将习得大量剑术技能。以下为部分技能预览。

基础技巧
横斩（Slash）
基础的下弧线挥砍攻击。

→ 指令：
突刺（Thrust）
简单的直线剑刺攻击。速度比普通横斩快，但威力更低。

→ 指令：
广域横斩（Wide Slash）
胸部高度的大范围横斩，可大幅向前突进。威力强劲，但使用后有短暂硬直。

→ 指令：
弧月斩（Arcing Slash）
剑刃向上挥出超过头顶的弧线攻击。对空中敌人的伤害高于普通攻击。

→ 指令：
回身突刺（Snap Thrust）
快速转身并向后突刺。该技能可与普通突刺快速衔接使用。

→ 指令：
侧翻（Cartwheel）
向远离敌人的方向侧翻，躲避攻击。

→ 指令：
翻滚（Roll）
向敌人方向或穿过敌人身体翻滚，躲避攻击或出其不意发动突袭。

→ 指令：
跳跃踢击（Jumpkick）
跳跃踢击适用于拉近与敌人的距离，或在徒手状态下使用。

→ 指令：


进阶技巧（随等级解锁）
下刺斩（Downstab）
经典终结技：跳跃至空中后，以剑刃朝下的姿态猛砸地面。

→ 指令：空中状态下，
连击横斩（Combo Slash）
横斩后可立即衔接的第二次攻击，剑刃向上挥击并向前小幅突进。

→ 指令：横斩后，
连击广域横斩（Combo Wide Slash）
普通横斩后可衔接广域横斩的组合攻击。

→ 指令：横斩后，
轮斩（Wheel Slash）
翻滚式横斩攻击，从下方击中敌人。可像翻滚一样穿过敌人身体。

→ 指令：
旋斩（Twirling Slash）
后空翻式挥砍，剑刃向上并向自身后方划出弧线。

→ 指令：
骑士突刺（Knight's Thrust）
向前冲锋，将剑当作长枪使用，宛如骑马的骑士。可贯穿多个敌人和防御。

→ 指令： 向前奔跑时，
满月斩（The Full Moon）
结合向前空翻与旋转横斩的强力剑术。对空中敌人有奇效。
