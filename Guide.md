# 2. 英语/日语版本汉化教程
汉化操作分两种场景，根据你的游戏来源选择即可，步骤简单易上手：

## 场景1：直接玩社区汉化版（无需Steam，不计入游戏时长）
如果下载的是上文提到的「圆桌汉化版（ver1.1）」，解压后直接打开汉化版安装目录中的 `sotes.exe` 启动文件，就能直接游玩汉化后的游戏，无需额外配置。

## 场景2：Steam正版汉化（计入游戏时长）
### 第一步：先备份游戏目录（关键！防止操作出错导致游戏损坏）
- 方法1（Steam内操作）：打开Steam客户端 → 找到《Fortune Summoners》 | 《フォーチュンサモナーズ -アルチェの精霊石-》→ 点击右上角「⚙️设置图标」→ 选择「管理」→ 点击「浏览本地文件」，打开游戏文件夹后，复制整个文件夹到其他位置备份。
- 方法2（直接查找文件夹）：打开电脑文件管理器 → 定位到Steam安装目录/你设置的其他磁盘的SteamLibrary库路径 → 进入 `steamapps\common\Fortune Summoners`（日语版根据实际情况查找一下，这边的测试版本是英文版） 路径 → 复制整个文件夹备份。

### 第二步：覆盖汉化文件
- 若使用「圆桌汉化版」完整包：删除Steam游戏目录下的所有文件，将汉化版的全部文件复制到该目录，之后通过Steam启动游戏即可（正常计入游戏时长）。
- 若仅有汉化补丁（非完整游戏包）：将补丁中的 `sotes.exe`、`sotesp.dll`、`sotesd.dll`、`sotesw.dll` 四个文件，复制到Steam游戏目录并覆盖原有文件。**注意：需确保汉化补丁版本与游戏本体版本匹配，版本不兼容可能导致游戏无法启动**。
- 补充语音包（可选）：如果你的游戏版本无语音，想要添加语音功能，额外将 `sotesx_s.dll`、`sotesx_d.dll` 两个文件复制到游戏目录即可。

# 3. 游戏高清与兼容性补丁（解决Win10/11适配问题）
这款游戏发行年代较早，在Win10/11系统上可能出现全屏异常、画质模糊、闪退、卡顿等问题，安装以下补丁即可有效解决：

### 第一步：下载补丁
访问 [DDrawCompat Github Releases 页面](https://github.com/narzoul/DDrawCompat/releases)，下载最新版压缩包（优先选择不带「debug」标识的普通发行版，无需复杂配置）。

### 第二步：安装补丁
解压下载的压缩包，找到其中的 `ddraw.dll` 文件，直接复制到游戏根目录（即与游戏启动文件 `sotes.exe` 同级的文件夹）。

### 补充说明
- 该补丁的作用是修复老游戏在新系统上的兼容性和画质问题，无需额外设置，复制后直接启动游戏即可生效。
- 实测验证：Windows 11 专业工作站版 25H2（版本号26200.7462）搭配 0.7.0 版本补丁（2025年12月17日发布），无需任何配置即可正常运行。

### 常见问题排查
- 若补丁无效，先查看 [DDrawCompat 官方文档](https://github.com/narzoul/DDrawCompat/) 查找解决方案。
- 尝试兼容性模式：右键 `sotes.exe` → 选择「属性」→ 切换到「兼容性」选项卡 → 勾选「以兼容模式运行这个程序」→ 选择「Windows Vista」或更早版本，可搭配「简化颜色模式」「以管理员身份运行此程序」等选项调试，修改后重启游戏。
- 以上方法均无效时，建议前往百度贴吧、Steam社区等平台求助。**本项目仅负责手册改造，不处理游戏本身的运行问题，请勿在项目Issue中反馈此类问题**。
